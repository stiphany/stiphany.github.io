Defect Detection Tool

Automated defect detection for optical profilometry data using machine learning.

Usage:

1. Upload DATX file(s) or select a demo
Tip: Select multiple files for batch viewing
2. Click "Run Inference" to process
3. Adjust percentile to highlight defects

Navigation: Use arrow keys (← →) or A/D to switch between loaded files. Click thumbnails at the bottom to jump to specific files.

Views:

3D: Point cloud visualization of surface data
Intensity: Single intensity channel view
All Data: Grid view of all datasets found in DATX file

Red highlights indicate detected defects.

Height Scaling:

Auto mode: Automatically scales based on data min/max
Manual mode: Set custom min/max values for consistent scaling across datasets
Scale multiplier: Adjust the vertical exaggeration (default: 20x)
